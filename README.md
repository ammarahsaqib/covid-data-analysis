# covid-data-analysis
Interactive COVID-19 data analysis dashboard built with Python, Pandas, Matplotlib and Plotly
An interactive data analysis project tracking COVID-19 case trends across 5 countries using Python.

## Countries Analyzed
- United States
- United Kingdom
- India
- Brazil
- Germany

## Tools Used
- Python
- Pandas (data cleaning & analysis)
- Matplotlib (static charts)
- Plotly (interactive dashboard)

## What This Project Does
- Loads and cleans a 162MB real-world dataset from Our World in Data
- Filters and analyzes daily new case trends across 5 countries
- Generates a static line chart saved as PNG
- Builds an interactive HTML dashboard you can hover over

## Data Source
Our World in Data — ourworldindata.org/covid-cases

## How to Run
1. Install dependencies: `pip install pandas matplotlib plotly`
2. Download the dataset from Our World in Data
3. Run: `python analysis.py`
